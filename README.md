# PhonePe Transaction Insights Dashboard

## Project Overview

This project is an interactive Power BI dashboard built using PhonePe transaction data. The dashboard provides insights into transaction performance, payment status, user behavior, service-wise transaction value, age segment contribution, and weekday vs weekend usage patterns.

The objective of this project is to transform raw transaction data into meaningful business insights that can help stakeholders understand customer payment trends and service performance.

## Tools & Technologies

- Power BI
- Microsoft Excel
- Power Query
- DAX
- Data Cleaning & Transformation
- Data Visualization

### Key Performance Indicators (KPIs)
- Total Transactions: 300K
- Total Transaction Value: ₹3.47 Billion
- Unique Users: 108K
- Successful Transaction Rate: 96%

### Interactive Filters
- Month Filter
- Payment Status Filter

### Visualizations
- Transaction Trend Analysis
- Age Segment Contribution
- Service Transaction Value Analysis
- Top 5 Users by Transaction Value
- Weekday vs Weekend Usage Analysis
- Business Insights Summary

## Business Insights

- Weekday transactions contribute more than 70% of total transactions.
- Loan services generate the highest transaction value among all services.
- Gen X and Millennials contribute the largest share of transactions.
- Transaction success rate remains consistently high at 96%.
